# Developer Guide

Welcome to the Developer Guide! This document provides an overview of the development process, tools, and best
practices for contributing to our project.

## Prerequisites

To run the project, you will need the following:

* [Docker](https://www.docker.com/)
* [Mise](https://mise.jdx.dev) (Improves the Quality of Life as a developer)
  * If not, these tools must be installed;
    * [Python](https://www.python.org) (3.12 or newer)
    * [uv](https://docs.astral.sh/uv/) (0.10 or newer)
    * [pre-commit](https://pre-commit.com/) (4.2 or newer)
      * Alternatively [prek](https://github.com/j178/prek) (used by default with mise)
    * [Node.js](https://nodejs.org) (22 or newer) in `web/`
    * [Yarn Classic](https://classic.yarnpkg.com/) (1.x) in `web/`
